## Installation

1. Clone this repo
```
$ git clone https://github.com/q82419/ntuchorus-server.git
```

2. Prepare NodeJS packages
Use `npm` to install the needed packages described in `package.json`.
```
$ cd ./your-working-directory/
$ npm install
```
